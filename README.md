# SkillCraft Web Development Task 1

## Responsive Educational Landing Page

This project is a responsive educational landing page built using HTML, CSS, and JavaScript as part of the SkillCraft Technology Web Development Internship.

### Features
- Fixed navigation bar
- Interactive hover effects
- Navbar changes style on scroll
- Responsive design for mobile, tablet, and desktop
- Educational website layout

### Technologies Used
- HTML5
- CSS3
- JavaScript
